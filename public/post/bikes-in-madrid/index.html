Bikes in Madrid

Oct 31, 2018 5 min read

As other cities in Europe, Madrid is becoming more bike friendly. At the same time, the city council is commited with open data and has been releasing a lot of information here about several issues in the city. A service that has captured my attention is BiciMad, the public bikes that can be used within the metropolitan area.

Data

There are two main datasets associated with BiciMad. Information on users since 2014 and inf

I begin with the information associated with bike docks corresponding from BiciMad. These data are hosted directly on the website of the Madrid’s city council. I load the data with import.

Where are the public bikes?

The first thing is to know where the bikes are. Madrid offers a public service. This information is obtained from the dataset corresponding to the bike stations.

incidencias = read.csv("https://datos.madrid.es/datosabiertos/TRAFICO/BICIMAD/2018/03/bici_incidencias20180315.csv", sep = ";")

places = import("https://datos.madrid.es/egobfiles/MANUAL/208327/bases_bicimad.csv") 

# Modify accents and ñ 

places = places %>%
  mutate(barrio = gsub('\xdc', 'U', Barrio), 
         barrio  = gsub("\xd3", "O", barrio),
         barrio  = gsub("\xd1O", "NO", barrio),
         barrio  = gsub("\xda", "U", barrio),
         barrio  = gsub("\xcd", "I", barrio),
         barrio  = gsub("\xd1", "N", barrio),
         barrio = gsub("\xc9", "E", barrio),
         barrio = gsub("[[:digit:]]", "", barrio),
         barrio = gsub("- ", "", barrio),
         distrito = gsub("\xcd", "I", Distrito),
         distrito = gsub("\xc1", "A", distrito))

# Modify name of variables



# Dataset with summary bikes
  
  bicis_barrio = places %>%
        group_by(barrio, distrito) %>%
        summarise(bicis = sum(`N\xfamero de Plazas`)) %>%
        arrange(desc(bicis)) %>%
        ungroup() %>%
        mutate(var_mean = ifelse(bicis > mean(bicis), "Above average", "Below average"),
               distrito = gsub("[[:digit:]]", "", distrito)) 


# Plot 
  
 bicis_barrio %>%
      ggplot(aes(reorder(barrio, bicis), bicis, color = distrito)) +
      geom_segment(aes(xend = reorder(barrio, bicis)), yend = 0, color = 'grey50') +
      geom_point(size = 3) +
      coord_flip() +
      labs(y = "Bikes", x = "Areas", title = "Location of bikes in areas of Madrid", subtitle = "", caption = "source: https://datos.madrid.es/") +
       theme_ipsum() +
      scale_color_viridis(discrete = T, name = "District", option = "viridis") +
      theme(panel.grid.major.x = element_blank(),
            panel.grid.minor.x = element_blank(),
            panel.grid.major.y = element_line(color = 'grey60', linetype = 'dashed'),
            axis.text.x = element_text(size = 14))
